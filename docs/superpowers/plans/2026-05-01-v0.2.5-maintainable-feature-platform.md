# v0.2.5 Maintainable Feature Platform Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build v0.2.5 as a maintainable feature platform with registry-backed commands/config/targets/capabilities, complete subscription lifecycle commands, richer diagnostics, one protocol-field extension slice, and release validation.

**Architecture:** Add plain C++17 registry structs as compile-time source-of-truth tables, then migrate command behavior to services and focused command modules. Keep `src/main.cpp` as a thin entrypoint/router while service-layer APIs carry reusable logic and tests cover registry consistency, CLI contracts, and user workflows.

**Tech Stack:** C++17, CLI11, yaml-cpp, nlohmann/json, CMake/CTest, Bash smoke tests, GitHub Actions.

---

## File Responsibility Map

- `include/subcli/registry.hpp` / `src/registry.cpp`: command/config/target/template/capability registry structs and lookup helpers.
- `include/subcli/config_service.hpp` / `src/config_service.cpp`: typed config get/set/remove/list/validate operations backed by `ConfigKeyRegistry`.
- `include/subcli/subscription_service.hpp` / `src/subscription_service.cpp`: subscription filtering, import, export, check, prune, and batch edit business logic.
- `include/subcli/diagnostic_service.hpp` / `src/diagnostic_service.cpp`: structured doctor probes and JSON/text result model.
- `include/subcli/commands/config_command.hpp` / `src/commands/config_command.cpp`: config command execution wrapper.
- `include/subcli/commands/sub_command.hpp` / `src/commands/sub_command.cpp`: subscription command execution wrapper.
- `include/subcli/commands/doctor_command.hpp` / `src/commands/doctor_command.cpp`: doctor command execution wrapper.
- `src/main.cpp`: keep global option parse, command dispatch, existing export command until later extraction, and call new command modules.
- `src/cli_completion.cpp`: read command/config/target suggestions from registry helpers.
- `src/capability_matrix.cpp`, `src/capabilities.cpp`, `src/protocol_registry.cpp`: migrate duplicated support metadata into registry-backed helpers without changing public output.
- `tests/subcli_tests.cpp`: unit/component tests for registries and services.
- `tests/cli_smoke.sh`: end-to-end CLI tests for new commands and diagnostics.
- `docs/config-file.md`, `README.md`, `README.subcli.md`, `docs/capability-matrix.md`: docs updates for registry-backed behavior and new lifecycle commands.
- `CMakeLists.txt`: add new source files and update project version to `0.2.5`.
- `.github/workflows/release-validation.yml`: build/test/package validation workflow.

    const auto& keys = configKeyRegistry();

    return it == keys.end() ? nullptr : &*it;

    return it == targets.end() ? nullptr : &*it;

#include "subcli/config_service.hpp"

#include "subcli/registry.hpp"

#include <cstdlib>
#include <sstream>

namespace subcli {
namespace {

bool parseInt(const std::string& value, const std::string& key, int& out, std::string& error) {
    char* end = nullptr;
    const long parsed = std::strtol(value.c_str(), &end, 10);
    if (end == value.c_str() || *end != '\0') {
        error = "expected integer for config key " + key;
        return false;
    }
    out = static_cast<int>(parsed);
    return true;
}

bool parseLong(const std::string& value, const std::string& key, long& out, std::string& error) {
    char* end = nullptr;
    const long parsed = std::strtol(value.c_str(), &end, 10);
    if (end == value.c_str() || *end != '\0') {
        error = "expected integer for config key " + key;
        return false;
    }
    out = parsed;
    return true;
}

bool parseBool(const std::string& value, const std::string& key, bool& out, std::string& error) {
    if (value == "true" || value == "1" || value == "yes") {
        out = true;
        return true;
    }
    if (value == "false" || value == "0" || value == "no") {
        out = false;
        return true;
    }
    error = "expected boolean for config key " + key;
    return false;
}

std::string boolString(bool value) {
    return value ? "true" : "false";
}

} // namespace

std::string getConfigValue(const AppConfig& config, const std::string& key, std::string& error) {
    for (const auto& entry : listConfigValues(config)) {
        if (entry.key == key) return entry.value;
    }
    error = "unknown config key: " + key;
    return "";
}

bool setConfigValue(AppConfig& config, const std::string& key, const std::string& value, std::string& error) {
    if (!findConfigKeyDescriptor(key)) {
        error = "unknown config key: " + key;
        return false;
    }
    if (key == "tun") return parseBool(value, key, config.tun, error);
    if (key == "profile") { config.profile = value; return true; }
    if (key == "profile_path") { config.profilePath = value; return true; }
    if (key == "output_dir") { config.outputDir = value; return true; }
    if (key == "template_dir") { config.templateDir = value; return true; }
    if (key == "asset_dir") { config.assetDir = value; return true; }
    if (key == "parallelism") return parseInt(value, key, config.parallelism, error);
    if (key == "timeout") return parseInt(value, key, config.timeout, error);
    if (key == "retry") return parseInt(value, key, config.retry, error);
    if (key == "log_level") { config.logLevel = value; return true; }
    if (key == "fetch_max_bytes") return parseLong(value, key, config.fetchMaxBytes, error);
    if (key == "core_paths.mihomo") { config.mihomoPath = value; return true; }
    if (key == "core_paths.sing_box") { config.singBoxPath = value; return true; }
    if (key == "core_paths.xray") { config.xrayPath = value; return true; }
    if (key == "node_management.dedupe") return parseBool(value, key, config.dedupeNodes, error);
    if (key == "node_management.rename_template") { config.renameTemplate = value; return true; }
    if (key == "node_management.include_regex") { config.includeRegex = value; return true; }
    if (key == "node_management.exclude_regex") { config.excludeRegex = value; return true; }
    if (key == "node_management.sort_by") { config.sortBy = value; return true; }
    if (key.rfind("templates.", 0) == 0) {
        const auto targetStart = std::string("templates.").size();
        const auto slotPos = key.find('.', targetStart);
        if (slotPos == std::string::npos) { error = "expected templates.<target>.<normal|tun>"; return false; }
        const auto target = key.substr(targetStart, slotPos - targetStart);
        const auto slot = key.substr(slotPos + 1);
        if (slot == "normal") { config.templateNormal[target] = value; return true; }
        if (slot == "tun") { config.templateTun[target] = value; return true; }
        error = "expected templates.<target>.<normal|tun>";
        return false;
    }
    if (key.rfind("grouping.region_rules.", 0) == 0) { config.regionRules[key.substr(22)] = value; return true; }

SubscriptionPrunePlan planPruneSubscriptions(const std::vector<Subscription>& subscriptions, bool disabled, int failedDays) {
    SubscriptionPrunePlan plan;
    for (const auto& sub : subscriptions) {
        const bool removeDisabled = disabled && !sub.enabled;
        const bool removeFailed = failedDays > 0 && !sub.lastError.empty() && sub.lastSuccess.empty();
        if (removeDisabled || removeFailed) {

        } else {

}

#include "subcli/diagnostic_service.hpp"

#include "subcli/registry.hpp"

#include <nlohmann/json.hpp>

namespace subcli {
namespace {

std::string severityName(DiagnosticSeverity severity) {
    switch (severity) {
        case DiagnosticSeverity::Info: return "info";
        case DiagnosticSeverity::Warning: return "warning";
        case DiagnosticSeverity::Error: return "error";
    }
    return "error";
}

void addFinding(DiagnosticReport& report, DiagnosticFinding finding) {
    if (finding.severity == DiagnosticSeverity::Error) report.hasError = true;
    report.findings.push_back(std::move(finding));
}

} // namespace

DiagnosticReport buildDiagnosticReport(const AppConfig& config, const std::vector<Subscription>& subscriptions, const std::string& workspaceSource) {
    DiagnosticReport report;
    addFinding(report, {"workspace.resolved", DiagnosticSeverity::Info, "workspace", "workspace source resolved", workspaceSource});
    for (const auto& key : configKeyRegistry()) {
        addFinding(report, {"config.key.registered", DiagnosticSeverity::Info, key.key, key.description, ""});
    }
    for (const auto& target : exportTargetRegistry()) {
        addFinding(report, {"export.target.registered", DiagnosticSeverity::Info, target.id, target.displayName + " target registered", target.outputFile});
    }
    if (config.profile.empty() && config.profilePath.empty()) {
        addFinding(report, {"profile.missing", DiagnosticSeverity::Error, "profile", "profile and profile_path are both empty", ""});
    } else {
        addFinding(report, {"profile.configured", DiagnosticSeverity::Info, "profile", "profile selection configured", config.profilePath.empty() ? config.profile : config.profilePath});
    }
    for (const auto& sub : subscriptions) {
        addFinding(report, {sub.enabled ? "subscription.enabled" : "subscription.disabled", DiagnosticSeverity::Info, sub.id.empty() ? sub.name : sub.id, sub.name, sub.cachePath});
        if (!sub.lastError.empty()) {
            addFinding(report, {"subscription.last_error", DiagnosticSeverity::Warning, sub.id.empty() ? sub.name : sub.id, sub.lastError, sub.cachePath});
        }
    }
    return report;
}

std::string diagnosticReportToJson(const DiagnosticReport& report) {
    nlohmann::json findings = nlohmann::json::array();
    for (const auto& finding : report.findings) {
        findings.push_back({{"code", finding.code}, {"severity", severityName(finding.severity)}, {"subject", finding.subject}, {"message", finding.message}, {"path", finding.path}});
    }
    return nlohmann::json{{"ok", !report.hasError}, {"findings", findings}}.dump(2);
}

std::string diagnosticReportToText(const DiagnosticReport& report) {
    std::string out;
    for (const auto& finding : report.findings) {
        out += severityName(finding.severity) + ": " + finding.code + " " + finding.subject + " " + finding.message + "\n";
    }
    return out;
}

} // namespace subcli
